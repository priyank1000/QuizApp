To build and run the application (in my machine), you need to do the following steps:

1) The Application is located in C:\cygwin\home\PRIYANK\sec\QuizApp
2) cd cygwin\home\PRIYANK\sec\QuizApp
3) gradlew build
4) gradlew collectQuizPlugin
5) gradlew collectQuestionPlugin
6) java -jar Core/build/libs/Core.jar
7) Enter "Core/build/libs/EngineeringQuiz.class" in the Main Application
